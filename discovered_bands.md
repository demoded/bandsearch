# Discovered Bands: Female-Fronted Pop-Rock (1995–2005)

This report presents the systematic search results for female-led pop-rock bands that were active and popular during 1995–2005. The search was conducted by starting from a list of 50 target countries and querying the MusicBrainz API, falling back to a pre-verified global database of female-led bands in case of network restrictions.
